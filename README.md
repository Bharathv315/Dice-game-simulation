# Dice-game-simulation
Java Dice Game

A simple and interactive Dice Game built using Java. This project demonstrates object-oriented programming concepts along with a basic graphical user interface (GUI) for user interaction.
Project Structure

- `Dice.java` → Handles dice rolling logic
- `Player.java` → Represents player data and score
- `GameEngine.java` → Controls game flow and rules
- `DiceGameUI.java` → GUI for user interaction
